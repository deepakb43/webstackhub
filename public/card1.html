HTML Tutorial

Learn the foundation of web development with HTML

  • Introduction
  • Basic Syntax
  • HTML Elements
  • Attributes
  • Page Structure

Introduction

HTML (HyperText Markup Language) is the standard markup language used to create the structure of web pages. It defines elements like headings, paragraphs, links, images, and more.

Basic Syntax

HTML consists of tags wrapped in angle brackets. Example:

<p>Hello World</p>

HTML Elements

Elements are the building blocks of HTML. Common ones include:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (link)
  • <img> - Image
  • <div> - Division/Container

Attributes

Attributes provide additional information about elements. For example:

<a href="https://webstackhub.com">Visit Us</a>

HTML Page Structure

A basic HTML page structure includes:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Heading</h1>
    <p>Paragraph</p>
  </body>
</html>
